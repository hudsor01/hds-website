| T-12-04 | Information Disclosure | logger sink (url/userAgent may contain tokens or PII) | mitigate | `logger.error` runs `redactSensitive` on metadata at the sink before console/DB/webhook; no raw secret is persisted unredacted. |
| T-12-05 | Information Disclosure | route response (leaking parsed body / stack to caller) | mitigate | Return `204` (no body) on success and a bare `400` on failure; never echo `parsed.data` or internal error text. The client only reads `res.ok`. |
