Per the patterns in 63-RESEARCH.md, author the four shared components:

**brand-layout.tsx**: `<Html><Head><Preview>{preview}</Preview><Body><Container>{children}</Container></Body></Html>` wrapper. Body bg `#ffffff`, system font stack. Container max-width 600px, centered, 24px padding.

**brand-heading.tsx**: takes `level: 1 | 2 | 3` and `children`, renders `<Heading as={`h${level}`}>` with BRAND.primary color, sensible font sizes (24/20/16px), margin-bottom 16px.

**brand-button.tsx**: takes `href` and `children`, renders `<Button>` with BRAND.primary background, white text, 12/24 padding, 6px radius, no underline.

**brand-footer.tsx**: takes optional `unsubscribeUrl` (when provided, render an unsubscribe link). Always renders contact info (email, address) using BUSINESS_INFO from `@/lib/constants/business`. Subtle text style (`#6b7280`, 12px). Top margin 32px.

- Imports from `'react-email'` (NEVER from `@react-email/components` or `@react-email/render`)
- Imports `BRAND` from `'@/lib/_generated/brand'`
- Has a TS interface for props
- Is the default export OR named export consistent with project convention (named recommended)

Refer to 63-RESEARCH.md "Recommended Component Pattern" for code shape.
