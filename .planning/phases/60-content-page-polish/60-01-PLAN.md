Make three targeted changes to `src/app/services/page.tsx`:

**1. Remove `'use client'` directive** (line 1)
The Services page has no useState, useEffect, event handlers, or browser APIs. The Card variant="service" is a client component, but Next.js handles rendering a client child from a server parent automatically — the page does NOT need 'use client'. Remove it.

**2. Add `export const metadata: Metadata`**
Import `type { Metadata }` from 'next' and `{ SEO_CONFIG }` from '@/utils/seo'. Pattern mirrors about/page.tsx:
```typescript
import type { Metadata } from 'next'
import { SEO_CONFIG } from '@/utils/seo'

export const metadata: Metadata = {
  title: SEO_CONFIG.services?.title || 'Our Services | Hudson Digital Solutions',
  description: SEO_CONFIG.services?.description || 'Custom web development, integrations, and business automation for growing businesses.',
  keywords: SEO_CONFIG.services?.keywords || [],
  openGraph: {
    title: SEO_CONFIG.services?.ogTitle ?? SEO_CONFIG.services?.title ?? 'Our Services | Hudson Digital Solutions',
    description: SEO_CONFIG.services?.ogDescription ?? SEO_CONFIG.services?.description ?? 'Website development, tool integrations, and workflow automation.',
    url: SEO_CONFIG.services?.canonical || '',
  },
  alternates: {
    canonical: SEO_CONFIG.services?.canonical || ''
  },
  robots: {
    index: true,
    follow: true,
    'max-snippet': -1,
    'max-image-preview': 'large',
    'max-video-preview': -1
  }
}
```
