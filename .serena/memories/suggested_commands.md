# Suggested commands

- bun run dev (local dev server)
- bun run build (production build)
- bun run start (run built app)
- bun run lint (ESLint)
- bun run typecheck (TypeScript)
- bun run test:unit (Bun tests)
- bun run test:e2e or bun run test:e2e:fast (Playwright)
- bun run generate-sitemap (sitemap)
- bun run env:setup and bun run env:validate (env helpers)
- bun run optimize:images (image optimization)
- bun run clean / bun run clean:test (cleanup)

# Common system utilities (Darwin)
- ls, cd, pwd
- rg (ripgrep), find
- git status, git diff, git add, git commit
